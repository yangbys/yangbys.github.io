数字滚动特效

今天想在数字上做点手脚，哈哈，让静态的数字动起来…这样页面就不会太死了。 也可以用于做老虎机抽奖活动等等

开始前，我们要准备两个工作
1、由于一些浏览器不兼容jquery animate => backgroundPosition 属性，所以要加载外部一个兼容脚本 animateBackground-plugin.js
2、要好看就得把数字替换成图片的方式 0 - 9 打竖排下来…
(如图)

直接上代码

function setNum(obj,num){
	//console.log('Nub',num)
	var numArr=num.toString().split('');
	var html='';
	for(var i=0; i<numArr.length; i++){
		html +='<i></i>';
	}
	obj.html(html);
	var span=obj.children('i');
	var h=20;
	var m=12;
	$.each(span,function(i,n){
		setTimeout(function(){
			$(n).animate(
				{backgroundPosition:"0 "+(-h*m*10 - h*numArr[i])+"px"},
				{
					duration:2000+i*500,
					easing: 'easeOutElastic',
					complete:function(){
						if(i==numArr.length-1){
							//console.log('end')
						}
					}
				}
			)
		},i*500)
	})
}
